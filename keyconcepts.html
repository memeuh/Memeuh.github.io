Key Concepts in SEO

PPC

PPC stands for pay per click. It is a form of digital advertising where businesses display ads on search engine or social media websites and only pay when someone clicks on the ad. When a user searches certain keywords, advertiser's ads will appear at the top of the search engine's results.

SEM

SEM strands for search engine marketing. It is a type of digital marketing that is a combination of SEO and PPC. SEO is organic interactions and PPM is paid for interactions. Search marketing is a blend of the two in order to optimize the number of people that see and interact with a website.

Crawling

Crawling is how a search engine or other company uses a program to search the internet. The program "crawls" a website and gathers it's information.

Indexing

Indexing is what happens after a website is crawled. The website's data is processed and collected. The data collected is then analyzed and grouped together based on similarity. This information is then displayed by the search engine when it recieves a query from the user.

Sitemap

A sitemap is a file that the website owner creates to provide information about the site's text, images, and videos. It tells the search engine which pages and files are most important on a site. It's a way for the search engine to crawl a website more effectively. This file is mainly used for big websites with over 500 pages.

Structured Data

Structured data is data that has been organized. It's adding additional html tags that the user can't see but a search engine can. It gives the search engine a better understanding of what the website is covering and the data is displaying.
